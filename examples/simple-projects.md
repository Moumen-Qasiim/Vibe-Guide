### Project 1: Tic-Tac-Toe (Console Game)

**The Concept:** A text-based game played in the terminal. No graphics, just text.

**Requirements:**

1. Draw a 3x3 grid on the screen using text characters (like `X`, `O`, and `-`).
2. Ask Player 1 to enter a number to place their `X`, then ask Player 2 for `O`.
3. Stop the game and print a message if a player gets three in a row, or if the board is full (a tie).

* **Pipeline Focus:** The Reviewer agent should strictly check that the game doesn't crash if a user types a letter instead of a number.

### Project 2: Personal Profile (Web Frontend)

**The Concept:** A single, static HTML page that acts as a digital business card.

**Requirements:**

1. Create a webpage with a main heading showing a person's name and a paragraph for a short bio.
2. Create a bulleted list of three hobbies or favorite movies.
3. Add a "Contact Me" button that simply pops up a basic web alert saying "Thanks for clicking!"

* **Pipeline Focus:** The Reviewer agent should strictly check for missing HTML tags, proper structure, and basic readable CSS colors.

### Project 3: The "Tap Counter" (Mobile)

**The Concept:** The most basic mobile app possible—a screen with a number and a button to increase it.

**Requirements:**

1. Display a large number `0` in the center of the mobile screen.
2. Add a large "+" button below the number. Every time it is tapped, the number increases by 1.
3. Add a smaller "Reset" button that changes the number back to `0`.

* **Pipeline Focus:** The Reviewer agent should check that the UI actually updates when the variable changes (basic state management).
